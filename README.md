# Sunset Cafe — site de prezentare

Site static cu 4 pagini și interfață în română, engleză și rusă.

## Pornire locală

```bash
python3 -m http.server 4173
```

Apoi deschide `http://localhost:4173`.

## Conținut

- `index.html` — pagina principală
- `menu.html` — meniul de mic dejun
- `poveste.html` — prezentarea cafenelei
- `contact.html` — date de contact, hartă și formular

Fotografia fațadei și imaginea meniului provin din galeria publică Google Maps a locației indicate. Imaginea ambientală cu cafea și croissant este un vizual editorial generat pentru site.
